# Vuejs-Weather-APP
Weather App build with Vue.js and call for OpenWeatherMap API

## Build Setup

cd my-app <br>
npm install <br>
npm run serve

## Modify or add API Key

Apply API key at https://openweathermap.org/api <br>
Then modify the App.vue at line 70 where "APPID = ur apikey"
